Welcome to HTML!

HTML (HyperText Markup Language) is the standard language for creating and designing web pages. It provides the basic structure for organizing content on the web.



Html Basics


Use of image

The <img> tag is used to embed images in an HTML document. It is a self-closing tag and requires the src attribute to specify the image file path and the alt attribute to provide alternative text for accessibility.

Paragraphs

The <p>tag is used to define paragraphs in HTML. It is a block-level element that separates text into distinct blocks. It's commonly used to structure and organize textual content.

Links

The <a> tag is used to create hyperlinks, allowing users to navigate to other web pages. It requires the href attribute to specify the destination URL . You can use the target="_blank" attribute to open the link in a new tab.

Lists
Unorded list
The <ul>(Unordered List) tag is used to create a list of items without any specific order. Each item in the list is defined by the <li> (List Item) tag. By default, unordered lists display with bullets preceding each item.
  • Item 1
  • Item 2
  • Item 3

Ordered list

The <ol>(Ordered List) tag is used to create a list of items with a specific numerical or alphabetical order. Similar to the unordered list, each item is defined by the <li> tag. Ordered lists display with numbers or letters by default.

  1. First item
  2. Second item
  3. Third item

Unstyled list

An unstyled list refers to a list that has its default styles removed. This is often achieved by applying custom styles or classes to <ul> or <ol> elements, altering their appearance. It's a way to create a list without the typical bullets or numbers associated with traditional lists.

Inline List

An inline list is a list where the list items ( <li>) are displayed horizontally instead of vertically. This is achieved by using CSS to set the display property of list items to inline or inline-block.

Formatting


<mark>:You can use the mark tag to highlight text.

<del>:This line of text is meant to be treated as deleted text.

<s>:This line of text is meant to be treated as no longer accurate.

<ins>:This line of text is meant to be treated as an addition to the document.

<u>:This line of text will render as underlined.

<small>:This line of text is meant to be treated as fine print.

<strong>:This line rendered as bold text.

<em>:This line rendered as italicized text.

<code>:This line will look like a code line.


Beware that those tags should be used for semantic purposes:

  • <mark> represents text which is marked or highlighted for reference or notation purposes.
  • <small> represents side-comments and small print, like copyright and legal text.
  • <s> represents elements that are no longer relevant or no longer accurate.
  • <u> represents a span of inline text which should be rendered in a way that indicates that it has a non-textual annotation.

If you want to style your text, you should use the following classes instead:

  • .mark will apply the same styles as <mark>.
  • .small will apply the same styles as <small>.
  • .text-decoration-underline will apply the same styles as <u>.
  • .text-decoration-line-through will apply the same styles as <s>.


Exibition


Blockquote

The <blockquote> element in HTML is used to represent a block of text that has been "quoted" from another source. It is commonly employed to indicate that the enclosed text is a citation, extract, or excerpt from a different document or source. When a browser renders text within a <blockquote> tag, it often adds indentation or other styles to visually distinguish the quoted content.


Code

The <pre> (preformatted) and <code>(code) elements in HTML are often used together to define a block of preformatted text, such as computer code. The <pre> element preserves both spaces and line breaks within the content, maintaining the original formatting. When used in conjunction with <code>, it signifies that the enclosed text is a piece of code.

                
                    // This is a code block
                    function greet(name) {
                        return "Hello, " + name + "!";
                    }
                    console.log(greet("World"));


Superscript and Subscript

The <sup> (superscript) and <sub> (subscript) elements in HTML serve to position text either above or below the baseline, respectively. Superscript is often used for notations, footnotes, or mathematical exponents, while subscript is commonly employed for chemical formulas or mathematical subscripts. The <sup> tag elevates the enclosed text, creating a smaller, raised appearance, while the <sub> tag lowers the text, producing a smaller, lowered appearance.

Subscript

Semantic Elements


Semantic element are elements with some meaning.

Semantic elements increase accessibility, improve code structure and make code more readable for both humans and machines.

<header>, <nav>, <main>, <article>, <section>, and <footer> are semantic HTML elements that provide meaning and structure to webpage content. They enhance clarity, accessibility, and SEO by accurately representing the content hierarchy. Using these elements improves the overall understanding and inclusivity of web documents.


Element Description
<header> The <header> element is used to represent the header of a section or a website. It typically contains logos, titles, and navigation elements.
<nav> The <nav> element is used to define the navigation section of a document, providing links to other pages or different sections of the same page.
<section> The <section> element is used to divide the content of an HTML document into thematic sections or groups of related content.
<aside> The <aside> element is intended to contain information that is related to the main content, such as sidebars, ads, or additional navigation elements.
<main> The <main> element represents the main content of an HTML document. It should only contain content directly related to the central content of the page.
<article> The <article> element is used to represent independent and standalone content, such as a blog post, a newspaper article, or a comment.
<footer> The <footer> element is used to represent the footer of a section or a website. It usually contains copyright information, contact details, additional links, and other similar data.


Tables


<table> is an essential element in HTML for organizing and presenting data in a structured manner on web pages. The <table> tag is used to create a table and can be complemented by various elements to define different aspects of the table's structure. The <thead> element is employed for the header section, typically containing column labels, while the <tbody> element represents the main body of the table where data is displayed. Each row in the table is defined by the <tr> tag, and within a row, <th> is used for header cells, and <td> is used for regular data cells. Additionally, the <caption> tag allows for the inclusion of a descriptive title for the table. Understanding these HTML table elements is crucial for web developers when designing and structuring information in a clear and organized way.



Element Description
<thead> Defines the header section of the table. Typically contains column labels.
<tbody> Defines the main section of the table body where data is displayed.
<tr> Defines a row in the table.
<th> Defines header cells in the table.
<td> Defines regular cells in the table.
<table> Defines the start of the table.
<caption> Provides a descriptive title for the table.
